자바스크립트가 뭐야

자바스크립트가 하는 일
  • 웹의 요소(엘리먼츠, 태그, 컴포넌트)
  • 웹 앱을 만들 때 활용할 수 잇어요!
  • 다양한 파생 라이브러리를 사용할 때 접근성이 좋아요! vue / react 등
  • 서버도 다룰 수 있게 됩니다. node.js
자바스크립트 프레임워크 종류
  • 웹 앱 : vue, react, angular
  • 서버 : node(호스팅 / 실시간 댓글, 소통, 삭제, 파일 관리, 보안 관리)
  • 그래픽 : d3
  • Dom : jaquery
자바스크립트는 어느 위치에 작성하나요?
  • CSS도 내부스타일 작성방식과 외부링크작성방식이 있듯이, JS도 내부스크립트 작성방식과 외부스크립트 작성방식이 있습니다.
  • HTML이나 CSS와 달리, JS는 대 소문자 구분이 잘 되어야 합니다.
  • CSS는 head안에 style태그를 열어서 사용하지요? js는 문서 어디에든 script~/script로 사용할 수 있습니다.
  • 삽입된 그 위치에서 바로 실행합니다.
  • script를 하나의 문서에서 여러 개 만들 수도 있습니다
  • script태그는 어디든 위치할 수 있다고 했지만, 미디어 요소(글자, 이미지같은)를 다루는 스크립트는 이미지, 글자 태그가 다 실행되고 난 다음의 위치, 즉 다루려는 태그위치보다 아래쪽에 위치해야 합니다. 보통은 그래서 대부분 body 끝나기 전에 몰아 배치하곤 합니다.
