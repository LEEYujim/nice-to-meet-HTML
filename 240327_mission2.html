HTML 기본 태그 미니테스트_(1)


01 다음 괄호 안을 채우시오.

  • h는 (heading제목)의 약자입니다.
  • p는 (paragraph문단,단락)의 약자입니다.
  • br은 (break line줄바꿈)의 약자입니다.
  • hr은 (horizontal수평선)의 약어입니다.
  • a는 (anchor링크를 검(닻))의 약자입니다.
  • href는 (hyper reference~데이터를 긁어온다 = 링크호출)의 약어입니다.
  • b는 (bold굵게)의 약자입니다.
  • i는 (italic기울임꼴)의 약자입니다.
  • sub는 (subscript아래첨자)의 약자입니다.
  • sup는 (superscript위첨자)의 약자입니다.
  • ins는 (insert강조; 기울임)의 약자입니다.
  • del은 (delete취소선)의 약자입니다.
  • ul은 (unordered list순서없는 목록)의 약어입니다.
  • ol은 (ordered list순서있는 목록)의 약어입니다.
  • li는 (list item목차 1개)의 약어입니다.
  • tr은 (table row테이블 행)의 약어입니다.
  • th는 (table header표에대한 머리,글)의 약어입니다.
  • td는 (table data표에 대한 위치 크기,한칸)의 약어입니다.
  • HTML 본문에 공백을 여러 번 넣고 싶을 때는 ( )를 사용합니다.
  • HTML 본문에 < 기호를 넣고 싶을 때는 (&#lt;)를 사용합니다.
  • HTML 본문에 > 기호를 넣고 싶을 때는 (&#gt;)를 사용합니다.
