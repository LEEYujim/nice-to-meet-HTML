① 선택자A > 선택자B-----ⓐ 자손 선택자
② 선택자A 선택자B-----ⓑ 후손 선택자

02 다음 질문에 OX를 표시하시오.

  1. 자손은 후손보다 넓은 개념입니다. (X)

  2. div > h1, h2는 “div 태그 아래”의 “h1 태그와 h2 태그”를 선택하는 선택자입니다. (X)

  3. CSS도 HTML처럼 형태로 주석을 사용합니다. (X)

  4. 2em은 2배를 나타내는 크기 단위입니다. (O)

  5. 색상 단위에서 투명도를 지정할 때는 a rgb( ) 등을 사용합니다.(X)

  6. URL을 지정할 때는 url ( ) 형태의 코드를 사용합니다. (O)

03 다음 코드에서 선택자가 어떤 태그를 선택하는지 설명하시오.

ex)div > a : div 태그 아래에 있는 a 태그를 선택한다.

  1. header a : header에 있는 모든 a를 선택한다.

  2. header > * > h1 : header에 바로 직계자손의 자손인 h1을 선택한다.(나의 딸의 손녀)

  3. header > div:first-child > a :header에 바로 직계자손의 첫번째 div의 직계 자손인 a 호출(나의 아들의 손주)

  4. header a:first-child > a : header의 후손 여러 a태그들 중 첫번째 a태그의 직계자손인 a태그

  5. div > h1, h2 : div의 직계 자손인 h1과, 일반 웹문서 전체를 통틀어 h2 태그 선택

  6. .title.card : class 별명이 title임과 동시에 class별명이 card 이기도 한 태그 선택

05 다음 코드에서 선택자가 어떤 태그를 선택하는지 적으시오.


ex) 1번 선택자 구문이 의미하는 바 (nav a) : nav 태그 아래 모든 a태그를 선택한다.
  • 1번 선택자 (header a) : header안에 있는 모든 후손 a태그를 선택한다.

  • 2번 선택자 (header > * > h1) :header에 있는 직계자손 안에 한번 더 타고 내려간 직계자손 h1을 선택한다.

  • 3번 선택자 (header > * > *:first-child > a) :header안에 있는 모든 후손 a태그를 선택한다.

  • 4번 선택자 (header a:first-child) : header에 있는 첫번째 태그를 선택한다.

06 다음 HTML 구조에서 음영으로 표시된 태그를 선택할 수 있는 선택자를 3개 이상 만드시오.
